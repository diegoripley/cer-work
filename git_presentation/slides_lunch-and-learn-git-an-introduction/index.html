What is Git?

  • Created in April 2005 by Linus Torvalds, creator of Linux.
    • Tailored for better speed, performance, flexibility, and usability than other version control systems.
  • It won. 2018 survey of developers by Stack Overflow found that 88.4% of 74,298 respondents used Git.
