# Background
- Regina was chosen as the ["Base Keystone" pipeline](https://web.archive.org/web/20201118184446/https://www.cer-rec.gc.ca/en/applications-hearings/view-applications-projects/keystone-xl/project-information.html) passes through the city and it was the only location that had high resolution DTM data.
- FCL Co-op Refinery Complex (in video) was confirmed via [publicly disseminated information](https://web.archive.org/web/20201114034807/https://www.cer-rec.gc.ca/en/data-analysis/energy-markets/provincial-territorial-energy-profiles/provincial-territorial-energy-profiles-saskatchewan.html) from Canada Energy Regulator.
